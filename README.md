# WukongSNS
基于 LeanCloud 来实现的 Swift 项目，练手和学习的类似微信朋友圈的一个应用。悟空的形象深入人性，暂且把这个项目叫悟空吧！数据来源于「名人朋友圈」这个 APP。

## 实现的功能：
- 浏览动态
- 切换角色
- 发表动态
- 评论
- 点赞
- 评论/赞的消息通知
